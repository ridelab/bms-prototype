# Project Structure

## Directories

```text
├─ docs             → documents
├─ src              → sources
│  ├─ main
│  │  ├─ resources  → main resources
│  │  └─ java       → main sources
│  └─ test
│     ├─ resources  → test resources
│     └─ java       → test sources
└─ target           → generated stuff
```
